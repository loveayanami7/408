## 指令寻址方式
### 指令寻址
1. 顺序寻址
2. 跳跃寻址
### 数据寻址
#### 格式
操作码 + 寻址特征 + 形式地址
#### 类型
1. 隐含寻址
	单地址指令格式约定第二个操作数由累加器ACC提供
2. 立即数寻址
	形式地址就是操作数本身
3. 直接寻址
	形式地址就是操作数的真实地址
4. 间接寻址
	形式地址是存操作数地址的存储单元的地址
5. 寄存器寻址
	形式地址是操作数所在寄存器的编号
6. 寄存器间接寻址
	形式地址是存操作数地址的寄存器的编号
7. 相对寻址🌟🌟🌟
	* PC + “1” + A，A为形式地址所表示的偏移量，补码表示
	* +1是PC自增，自增多少取决于**指令字长**（占多少个地址单元）
8. 基址寻址
	* 将基址寄存器BR的内容加上形式地址得到操作数所在地址
	* 基址寄存器面向操作系统，内容由操作系统/管理程序决定
	* 基址寄存器可用专用寄存器，也可用通用寄存器，用户可以指定某个通用寄存器为BR，但内容依然由操作系统指定
9. 变址寻址
	与基址寻址的不同在于其内容可由用户指定，是面向用户的
10. 堆栈寻址
	* 后进先出
	* 硬堆栈：寄存器组；软堆栈：划分一块主存
	* 地址由寄存器**堆栈指针SP**给出
## 汇编
### 主要寄存器

| 16bit | 32bit |   说明   |
| :---: | :---: | :----: |
|  AX   |  EAX  |  累加器   |
|  BX   |  EBX  | 基地址寄存器 |
|  CX   |  ECX  | 计数寄存器  |
|  DX   |  EDX  | 数据寄存器  |
|       |  ESI  | 变址寄存器  |
|       |  EDI  | 变址寄存器  |
|       |  EBP  | 堆栈基指针  |
|       |  ESP  | 堆栈顶指针  |
为实现向后兼容，EAX、EBX、ECX、EDX的高两位字节和低两位字节可以独立使用，AX、BX、CX、DX的高低字节又可分别作为两个8位寄存器（ah、al）
### Intel格式
1. 大小写不敏感
2. 第一个为目的操作数，第二个为源操作数，从右到左
3. 寄存器和立即数前不需要前缀
4. 用"[ ]"内存寻址
###  常用指令
#### 操作数标记
* `<reg>`：寄存器
* `<mem>`：内存地址
* `<con>`：常数
#### 机器指令
* x86中的指令机器码长度为1字节，**同一指令的不同用途有多种编码方式**
